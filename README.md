# Nuxt-Blog

Sample Blog service build by SPA

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install
```

## Development Server

Start the development server on `http://localhost:3000`

```bash
npm run dev
```

## Production

deploy

```
$ npm run deploy
```
